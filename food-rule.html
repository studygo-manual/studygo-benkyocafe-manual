📖 館内でのお食事について

勉強カフェは学習に集中いただくための共用空間です。
周りのお客様への配慮として、以下にご協力をお願いいたします。

1 🌸 匂い — 香りが広がるお食事はご遠慮ください
開封時に香りが周囲に広がるもの（カレー・エスニック・ニンニク料理・発酵食品 等）
2 🔥 温度 — 温かいお食事はご遠慮ください
湯気が立つもの・温め直したもの・デリバリーの温かい料理など（冷めたお弁当・サンドイッチ等はお召し上がりいただけます）
3 🔊 音 — 周囲への配慮をお願いいたします
お菓子類（館内販売のオフィスグリコ含む）はお召し上がりいただけます。周囲の学習に著しく支障がある場合のみ、スタッフよりお声がけする場合がございます

✅ お召し上がりいただけるもの（例）

サンドイッチ / おにぎり / パン類
お菓子・チョコレート・和菓子
ナッツ・ドライフルーツ
バナナ・りんご 等の果物
冷たいお弁当 / プロテインバー
お飲み物全般

❌ ご遠慮いただくもの（例）

カップラーメン・カップ麺類

温かいお弁当・デリバリー料理

🙏 上記に該当しないお食事については、スタッフがご案内いたします

ご不明な点はお気軽にスタッフまでお声がけください

🔧 スタッフ向け 運用の考え方

基本スタンス

  • 細かい品目リストは作らない（会員から穴を突かれるため）
  • 「基準」として3原則を示し、最終判断は現場に委ねる
  • 厳密な線引きより「周りのお客様への配慮」という観点で案内する

判断の目安（3軸で判断）

  • 🌸 匂い：開封時に周囲に匂いが広がる → NG / 匂いが広がらない → OK
  • 🔥 温度：温かい・湯気が立つ・温め直し → NG / 冷めている → OK
  • 🔊 音：原則OK（オフィスグリコでも販売しているため）。極端な場合のみ個別に配慮をお願いする

※ 3軸のうち匂いと温度のどちらかに該当するとNG。音は原則許容で例外対応。

よくある質問への回答

  • Uber Eats等のデリバリー → 温かい料理が多いため原則NG。冷たいサンドイッチ等は個別判断
  • 弁当の温め直し → 店内に電子レンジは無し。温かい状態で持ち込まれたものはNG
  • カップスープ・カップ麺 → 温かい＋液体ゴミ処理の負担でNG
  • ポテチ・せんべい → 原則OK（館内のオフィスグリコでも販売しているため）。著しく支障がある場合のみ個別対応
  • みかん → 基本OKだが匂いが気になる場合は個別案内

お客様への案内文例

恐れ入ります、共用スペースのため匂いの強いお食事はご遠慮いただいております。
パン類・おにぎり・お菓子などはお召し上がりいただけますので、お召し上がりの際はよろしくお願いいたします。

エスカレーション

  • お客様から「なぜこれがダメなのか」と問われた場合 → 「周りのお客様からお声をいただくため」と説明
  • 判断に迷う場合 → 無理にその場で結論せず、一度MGに確認する
  • トラブルに発展しそうな場合 → 個室ラウンジへのご案内も選択肢

📝 議論の経緯（MG陣記録）

当初は細かい品目リスト（OK 12品目 / NG 12品目 / グレー 10品目）で4名投票を実施しましたが、 以下の理由から基準方式に変更しました。

  • 細かくすると会員から「リストにない◯◯はどうなんだ」と穴を突かれる
  • 4名の意見が割れた項目が多く、全ての品目を決め切るのが困難
  • 現場判断の余地を残した方が柔軟に運用できる

投票で得られた重要な視点

  • 温度軸（ねもと・みっちー）：冷めたものは基本匂わない。温め直しは厳禁
  • 音問題（みっちー）：せんべい・ポテチは匂いより咀嚼音
  • デリバリー対応（棟元）：Uber Eats等の質問が必ず出る
  • 液体ゴミ問題（ねもと）：カップスープのゴミ処理が現場負担
