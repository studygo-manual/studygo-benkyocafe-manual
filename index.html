システム概要

スタディGOとは

スタディGOは、勉強場所を探しているユーザーと店舗をマッチングするアプリです。

ユーザーは以下をアプリ内で完結できます：

  • 店舗検索・比較
  • 内見予約
  • 契約・決済
  • QRコードでのチェックイン・チェックアウト

利用プランの種類

月額プラン

毎月定額で利用できるプラン。プラン時間を超過した場合は1分単位で課金されます。

ドロップイン（都度利用）

予約不要で利用できるスポット利用。チェックイン時に1日最大料金を請求し、チェックアウト時に精算します。

デフォルトプラン（60分プラン）

スタディGO独自のプラン。月額6,600円で毎日60分利用可能。超過分は1分単位課金。

ロッカー契約注意事項（ユーザー向け表示）

1. 利用に関して
ロッカーは店頭での手続きが必要です。スタッフ有人時間帯にご来店ください。

2. 鍵の管理
鍵の紛失・破損時には、再発行手数料(3,300円)が発生いたします。紛失時は速やかにスタッフまでお申し出ください。

ロッカー解約注意事項（ユーザー向け表示）

1. 荷物の撤去
解約日までに、ロッカー内の荷物をすべてお持ち帰りください。残置物がある場合は、翌月分の料金を請求させていただく場合があります。

2. 鍵の返却
解約日当日または指定日までに、受付まで鍵をご返却ください。鍵を紛失された場合は、再発行手数料(3,300円)が発生します。

管理画面ログイン

ログイン方法

  1. 運営からメールで送られてきたログイン情報を確認
  2. 管理画面にアクセス
  3. 「法人メールアドレス」と「パスワード」を入力
  4. ログインボタンをクリック

ポイント

  • パスワードは変更可能です
  • 運営側で法人情報を更新するとパスワードがリセットされます
  • 対応が必要なことはバッジと件数で表示されます

管理画面メニュー

  • ダッシュボード - 未返信メッセージ、本日の内見予約、新規契約者などを確認
  • 内見予約一覧 - 内見予約の管理
  • チェックイン実績 - ユーザーの利用履歴確認
  • 会員管理 - 会員情報の確認、メッセージ送信
  • 店舗管理 - 店舗情報の設定
  • 支払い実績確認 - 決済情報の確認
  • 営業日・営業時間管理 - カレンダー設定
  • 加盟法人情報 - 法人情報の確認

チェックイン・チェックアウト

事前準備

掲載開始前に、受付カウンターや入り口ドア等にQRコードを設置してください。

チェックインの流れ

  1. ユーザーがアプリでQRコードを読み取る
  2. チェックイン完了画面が表示される
  3. 必要があれば画面を確認

※QR読取エラー時は店舗コードでもチェックイン可能です（非推奨）

チェックアウトの流れ

  1. ユーザーがアプリでQRコードを読み取る
  2. チェックアウト完了・精算画面が表示される

重要

  • チェックアウトが行われるまで利用があったものとみなされます
  • 課金対象の場合は課金されます
  • 営業終了時間に強制チェックアウトが行われます

チェックアウト時の精算

  • 月額プラン時間超過分の算出・請求
  • ドロップイン利用時の1日最大請求額からの精算（返金または請求）
  • 超過時間は店舗設定の〇〇円/1分で計算（計50円未満は切り捨て）

利用履歴の確認方法

  1. 管理画面の「チェックイン実績」をクリック
  2. 店舗で絞り込み
  3. 日付で範囲指定

チェックアウト後は明細確認可能、チェックアウト前は時間編集可能です。

ドロップイン利用

ドロップイン利用が想定されるシーン

  • 月額利用を検討しているが、まずはお試しとして利用
  • 月額会員だが、プラン時間外にチェックインする場合
  • 月額会員だが、〇〇分プランの残時間が無い状態
  • 勉強場所を近辺で探していた人

ドロップイン利用の流れ

1. QR読取

ユーザーがQRを読み取り、ドロップインプラン選択画面が表示

2. 契約・チェックイン

チェックイン時にドロップインプランを契約（契約即チェックイン）。この時点で1日最大料金を請求

3. チェックアウト・精算

QRを読み取り、チェックアウト完了・明細画面が表示

月額会員のドロップイン利用

月額会員でも以下のノンストップ利用が可能です：

プラン時間前にドロップイン → プラン時間内に突入 → プラン時間外の超過利用

これを容認する場合、鍵付与の際にプラン時間内に限定せず、営業時間全ての鍵付与を行う必要があります。

内見予約対応

内見予約が入ったら

  1. 登録の店舗メールアドレスに予約のお知らせが届く
  2. 管理画面の「内見予約一覧」で確認
  3. ステータスを変更：
    • ◎内見可能の場合 → 即確定
    • △対応不明の場合 → 仮予約（「予約確定」または「予約キャンセル」に変更）
  4. ユーザーにメッセージを送信（推奨）

内見対応

  • 通常の内見対応と同じ内容でOK（営業時間や利用ルール等の説明）
  • ※店舗で提供されているプランの紹介・説明や、他サービスの紹介・説明はお控えください

内見後の契約

気に入っていただけたら「アプリを通じて」契約してもらいます。

  • ユーザーがアプリで単独で完結できるので、店舗側作業はありません
  • 契約についての質問は次項「契約時の対応」を参照

内見予約日時経過後

  1. 対応実績に合わせてステータスを変更
  2. 来店した場合 →「内見実施」
  3. 来店しなかった場合 →「内見キャンセル」
  4. ステータス変更後、一覧から非表示になりアーカイブされます

契約時の対応

店内で契約した場合

契約はユーザーがアプリで行うため、店舗側作業はありません。

契約後は通常の店舗でのご入会と同様の業務を行ってください：

  • 施設ルールの読み合せ
  • 無人時間帯の鍵付与
  • 身分証明書の確認、コピー等
  • 緊急連絡先を伝える（店舗利用に係る）

店外で契約した場合

その場にいないで契約したユーザーとは：

  • 内見をしたがその場で契約しなかったユーザー
  • ドロップインなどで過去に接点のあったユーザー
  • いきなりアプリから契約したユーザー

対応フロー

  1. 「メール」通知とダッシュボードの「新規契約したお客様」で確認
  2. 「メッセージ」またはメールアドレスでユーザーに連絡
  3. 初回利用前の所定手続き（ルール説明、鍵発行、身分証確認等）が必要な旨を伝える
  4. 来店できる日のアポイントメントを取得（利用開始初日以前が推奨）

月額プランの仕組み

契約時の請求
  • 入会金5,500円
  • 店舗別初期費用
  • 利用開始月分（日割り）
  • 翌月分月額料金
最低継続期間 利用開始月とその翌月の2ヶ月
解約申請 前月11日〜当月10日までの申請で当月末解約
自動更新 当月10日までに解約申請が無ければ、15日に翌月分請求。1日に1ヶ月単位で自動更新

プラン変更ルール等、詳細はメッセージ機能で「運営に聞いてください」でOKです。

解約時の対応

解約の流れ

  1. 10日までの解約申請で月末での契約終了 → 11日に一覧に解約者が追加
  2. ダッシュボードで解約者を確認
  3. 必要があれば連絡（物理キーの回収が必要な場合など）
  4. 最終利用日以降に鍵の権限削除など、店舗システム側で退会処理

管理画面での操作

  • 「鍵回収ステータス」で管理（未回収/回収済み）
  • 対応完了後は「確認済み」で一覧から非表示に

各種設定

内見予約カレンダーの設定

「営業日・営業時間管理」から設定します。

設定項目

  • 締め切り設定 - いつまで内見予約を受け付けるか（例：「前日の20:00まで」「当日の内見開始時刻の1時間前」など）
  • 基本内見可能時間 - 「◎内見可能」となる時間帯
  • 対応不明 - チェックを入れると「△対応不明」（仮予約）で表示
  • 対応不可 - チェックを入れるとその曜日の内見予約を停止

無人店舗の場合は基本は全て「△対応不明」とする運用が想定されます。

営業日・営業時間設定（月額会員向け）

スタディGOの月額会員の受け入れ可能時間・曜日を設定します。

  • 頻繁な変更は推奨しません
  • 現在提供中のプランと干渉しないようにご注意ください
  • 月額契約中ユーザーに事前連絡が必要な場合があります
  • 終了時間1分前にチェックイン中ユーザーの強制チェックアウトが行われます

営業日・営業時間設定（都度利用向け）

ドロップインの受け入れ可能時間・曜日を設定します。有人営業時間帯の設定が想定されます。

「月額会員向け」と「都度利用向け」は独立したカレンダーのため連動しません。両方の設定を変更し、整合させる必要があります。

店舗管理での設定

「店舗管理」→ 一番右の編集アイコンをクリック

チェックイン時表示 入室方法や暗証番号、利用ルールなどを記載
契約前同意事項 契約前にユーザーに同意を求める内容を記載
最短利用開始日 「n日以降」の営業日と設定可能
0=当日から契約して利用可
1=契約の翌日から利用可
2=契約の2日後から利用可

※内見対応時にその日から入会させたいときはその時のみ設定変更

トラブル対応

チェックインを忘れた場合

同日の場合
  1. その場でチェックインしてもらう
  2. 管理画面「チェックイン実績」→「編集」で現在のチェックインを修正
別日の場合 掲載店側では修正できないため、運営にご相談ください。
運営から追加請求を行います。

チェックアウトを忘れた場合

同日の場合
（店舗を離れてしまった）		 「QR読取」→「QRが読み取れない場合」→ 店舗コードを入力
※店舗コードはQRのポップに記載
別日の場合 営業時間終了時点で自動チェックアウトされています。
チェックアウト忘れの返金はありません。

間違ってチェックイン/チェックアウトした場合

返金や修正措置はありません。

店舗では対応できない旨を伝え、対応が必要な場合は運営に連絡してもらってください（アプリ内メッセージ機能）。

お休みや営業時間の変更

イベント貸切、店舗メンテナンス、年末年始など、数時間〜数日程度の変更の場合：

  1. 「会員管理」から月額契約中ユーザーにメッセージまたはメールで連絡
  2. 「営業日・営業時間管理」で「月額会員向け」と「都度利用向け」両方の設定を変更

※状況によっては返金を求められる可能性があります。掲載店に決定権がありますが、ご相談やサポートは運営で承ります。
