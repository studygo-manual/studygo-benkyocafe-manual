1. ファイル構成

マニュアルは以下のファイルで構成されています：

manual/
├── index.html　　　← マニュアル本体（文章を編集）
├── style.css　　　 ← スタイル（色・レイアウトを編集）
└── 編集ガイド.html　← このガイド
変更したい内容 編集するファイル
文章・項目の追加/変更/削除 index.html
色・フォント・余白などの変更 style.css

2. 編集に使うツール

推奨：Visual Studio Code（無料）

以下からダウンロードできます：
https://code.visualstudio.com/

Visual Studio Codeの利点
  • HTMLタグが色分けされて見やすい
  • 入力補完機能がある
  • エラーを教えてくれる

簡易：メモ帳（Windows標準）

Windowsに最初から入っているメモ帳でも編集できます。

3. ファイルの開き方

Visual Studio Codeの場合

  1. 1Visual Studio Codeを起動
  2. 2「ファイル」→「フォルダーを開く」
  3. 3manualフォルダを選択
  4. 4左のファイル一覧から編集したいファイルをクリック

メモ帳の場合

  1. 1index.htmlを右クリック
  2. 2「プログラムから開く」→「メモ帳」を選択
注意

index.htmlをダブルクリックするとブラウザで開きます。編集するには「右クリック」→「プログラムから開く」を使ってください。

4. HTMLの基本

HTMLは「タグ」で文章を囲んで構造を作ります。

<!-- これはコメント。画面には表示されません --> <p>これは段落です。</p> <h3>これは見出しです</h3> <ul> <li>リスト項目1</li> <li>リスト項目2</li> </ul>
タグ 意味 使用例
<p>...</p> 段落（文章） <p>説明文です。</p>
<h2>...</h2> 大見出し <h2>セクション名</h2>
<h3>...</h3> 中見出し <h3>小項目名</h3>
<ul>...</ul> 箇条書きリスト 番号なしリスト
<ol>...</ol> 番号付きリスト 手順など
<li>...</li> リスト項目 リストの各項目
<strong>...</strong> 太字・強調 <strong>重要</strong>

5. テキストを変更する

最も簡単な編集です。タグの中の文字を書き換えるだけです。

例：電話番号を変更する

変更前：

<p>電話：03-6302-0774（平日10:00〜18:00）</p>

変更後：

<p>電話：03-1234-5678（平日9:00〜17:00）</p>
検索機能を活用

Ctrl + F で検索窓が開きます。変更したい文字を検索すると、該当箇所がすぐ見つかります。

6. リスト項目を追加・削除する

項目を追加する

<ul> <li>既存の項目1</li> <li>既存の項目2</li> <li>新しく追加した項目</li> ← この行を追加 </ul>

項目を削除する

<li>から</li>までの1行をまるごと削除します。

7. セクションを追加する

新しいセクション（章）を追加する場合のテンプレートです。

<!-- 新しいセクション --> <section id="new-section"> <h2>新しいセクション名</h2> <div class="card"> <h3>見出し</h3> <p>説明文をここに書きます。</p> </div> </section>
目次への追加を忘れずに

新しいセクションを追加したら、サイドバーの目次にもリンクを追加してください。

<!-- #sidebarの中のulに追加 --> <li><a href="#new-section">新しいセクション名</a></li>

8. カード（囲み）を追加する

通常のカード

<div class="card"> <h3>カードのタイトル</h3> <p>カードの内容です。</p> </div>

重要なカード（緑の強調）

<div class="card important"> <h3>重要なお知らせ</h3> </div>

警告カード（赤の強調）

<div class="card warning"> <h3>注意事項</h3> <p>注意すべき内容です。</p> </div>

9. 表を編集する

<table class="info-table"> <tr> <!-- 1行目 --> <th>見出しセル</th> <!-- 見出し --> <td>データセル</td> <!-- データ --> </tr> <tr> <!-- 2行目 --> <th>営業時間</th> <td>10:00〜22:00</td> </tr> </table>

10. リンクを追加する

<!-- 外部サイトへのリンク --> <a href="https://example.com">リンクテキスト</a> <!-- メールリンク --> <a href="mailto:info@example.com">info@example.com</a> <!-- ページ内リンク（#の後ろはid名） --> <a href="#checkin">チェックインの項目へ</a>

11. 実践例：営業時間を追加する

「システム概要」セクションに勉強カフェの営業時間を追加する例です。

手順

  1. 1index.htmlを開く
  2. 2Ctrl + F で「システム概要」を検索
  3. 3該当セクションを見つけたら、以下のコードを追加
<!-- 「システム概要」セクション内に以下を追加 --> <div class="card"> <h3>勉強カフェ 営業時間</h3> <table class="info-table"> <tr> <th>平日</th> <td>6:00〜23:00</td> </tr> <tr> <th>土曜</th> <td>6:00〜22:00</td> </tr> <tr> <th>日祝</th> <td>6:00〜21:00</td> </tr> </table> </div>
  1. 4ファイルを保存（Ctrl + S）
  2. 5ブラウザで確認（index.htmlをダブルクリック、またはF5で更新）

12. バックアップについて

編集前に必ずバックアップを取ってください

manualフォルダをコピーして、日付付きの名前で保存しておくことをおすすめします。

例：manual_backup_20240205

13. トラブル対処法

レイアウトが崩れた

  • タグの閉じ忘れがないか確認（<div>には</div>が必要）
  • 引用符（"）の閉じ忘れがないか確認
  • バックアップから復元

変更が反映されない

  • ファイルを保存したか確認（Ctrl + S）
  • ブラウザをリロード（F5 または Ctrl + F5）
  • ブラウザのキャッシュをクリア

文字化けする

  • ファイルをUTF-8形式で保存してください
  • メモ帳の場合：「名前を付けて保存」→ 文字コードを「UTF-8」に変更
困ったときは

編集で困った場合は、バックアップから復元するか、このガイドを参考にしてください。大きな変更を加える前には必ずバックアップを取っておきましょう。
